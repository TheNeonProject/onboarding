# Installation

You need to install pipenv

```bash
pip3 install pipenv
pipenv --python 3.7
pipenv install
```

# Running

```bash
pipenv run python cli.py
```

**Or you can open the VS Code, go to the Debug Tab and click the play button to run the script**
